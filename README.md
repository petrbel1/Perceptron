Результаты: 
Модель успешно обучена. Функция ошибок при стандартном обучении падает почти до 0.3,

Выводы по экспериментам:
1) lr = 1 почти мгновенно обучает модель, после первых эпох результаты даже чуть-чуть ухудшаются, lr = 0.1 обучает более равномерно, к эпохе 100 дообучает до оптимального состояние, lr = 0.01 заметно недообучает модель, lr = 0.001 критически недообучает модель(L = 0.5)
2) Батчи - чем мельче, тем быстрей обучение, но слишком малые батчи вызывают переобучение
3) Веса - результаты для нулевых и маленьких весов весьма схожи, а вот большие замедляют обучение. Они снижают влияние непосредственно значений(небольших на фоне весов)
