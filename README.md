Модель: однослойный перцептрон. Обрабатывает данные по двум признакам, соответственно имеет 2 веса и b
Формула обработки признаков: z = w[1] * p[1] + w[2] * p[2] + b
Далее z передается в сигмоиду: y = 1 / (1 + e**-z)
Для получения из результатов сигмоиды бинарного результата(0 или 1) используется функция
 predict(округление в большую сторону от 0.5, иначе в меньшую)
 Функция потерь: (-sum на i в y_true)/i: 
 (y_true[i] * log(y_pred[i] + eps) + (1 - y_true[i]) * log(1 - y_pred[i] + eps)). y_true -реальные данные, e_pred - предполагаемая классификация, eps - сверхмалое число во избежание логарифма нуля и ошибок 
 Оюучение: на каждой эпохе для веса w: w = w - lr / len(data) * grad, где grad = sum((y_pred[i] - y_true[i]) * X_input[i])

Предобработка данных: данные генерируются при помощи sklearn.datasets.make_classification(). Далее с помощью sklearn выполняется стандартизация признаков и разделение на обучающие и тестовые выборки со стратификацией

Результаты: 
Модель успешно обучена. Функция ошибок при стандартном обучении падает до 0.4, 
accuracy доходит до 0.87 на обучающей и 0.89 на тестовой выборке(почему на тестовой лучше - неизвестно)
